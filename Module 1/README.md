# Module 1 – Front-end Cơ Bản (JavaScript) – Case Study

## Mô tả module
Module 1 là phần học **Front-end cơ bản tại CodeGym**, tập trung vào **JavaScript** và các kiến thức nền tảng để xây dựng **giao diện web tương tác**. Học viên sẽ thực hành qua các bài tập và case study, nắm chắc kiến thức về **biến, mảng, hàm, thao tác DOM và sự kiện**.

## Nội dung học
- **Biến & kiểu dữ liệu**: số, chuỗi, boolean, mảng.  
- **Hàm (Function)**: khai báo, tham số, trả về giá trị.  
- **Mảng (Array)**: tạo, truy xuất, xử lý dữ liệu mảng.  
- **Phương thức xử lý dữ liệu & chuỗi**: nối chuỗi, cắt chuỗi, tìm kiếm, lọc.  
- **Thao tác DOM**: tạo bảng, cột, hiển thị và thay đổi nội dung trang web.  
- **Sự kiện (Event)**: click, input, hover…  

## Hình thức kiểm tra
- **Thực hành (Practice)** – Giải các bài tập JavaScript.  
- **Lý thuyết (Theory)** – Trắc nghiệm kiến thức cơ bản.  
- **Audit** – Kiểm tra tổng hợp kiến thức và kỹ năng thực hành để pass module.  

## Kết quả
- Hiểu và sử dụng thành thạo JavaScript cơ bản.
