# Module 3 – AI Chatbot phân tích cảm xúc khách hàng

## Giới thiệu
Dự án này thuộc **Module 3 – AI** trong chương trình học Full-stack tại **CodeGym**.  
Mục tiêu của dự án là xây dựng một **chatbot tự động** có khả năng **phân tích cảm xúc của khách hàng** (tích cực, trung tính, tiêu cực) dựa trên dữ liệu cho sẵn và **tạo phản hồi phù hợp thông qua API AI**.

## Mục tiêu
- Phân tích nội dung bình luận của khách hàng.
- Xác định cảm xúc (sentiment) từ dữ liệu huấn luyện có sẵn.
- Tự động phản hồi bình luận dựa trên kết quả phân tích.
- Làm quen với việc tích hợp API AI vào ứng dụng.

## Chức năng chính
- Phân loại cảm xúc khách hàng.
- Tự động trả lời bình luận theo từng loại cảm xúc.
- Kết nối và sử dụng API AI để sinh nội dung phản hồi.
- Áp dụng cho các tình huống chăm sóc khách hàng cơ bản.

## Công nghệ sử dụng
- Ngôn ngữ: JavaScript, HTML, CSS, Python
- AI API: OpenAI 
- Dữ liệu: Tập dữ liệu bình luận khách hàng có sẵn
- Định dạng dữ liệu: JSON
- Giao tiếp: REST API

## Quy trình hoạt động
1. Nhận bình luận từ khách hàng.
2. Phân tích cảm xúc dựa trên dữ liệu cho sẵn.
3. Gửi thông tin cảm xúc đến AI API.
4. Nhận phản hồi và trả lời khách hàng.

## Kết quả đạt được
- Hiểu cách phân tích cảm xúc trong xử lý ngôn ngữ tự nhiên (NLP).
- Biết cách sử dụng và tích hợp AI API.
- Ứng dụng AI vào bài toán thực tế trong chăm sóc khách hàng.
